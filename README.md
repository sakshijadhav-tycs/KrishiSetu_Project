# 🌾 KrishiSetu – Farmer to Customer Marketplace

KrishiSetu is a full-stack web application developed to provide a direct digital
marketplace between farmers and customers. The platform helps farmers sell their
produce directly while customers get fresh agricultural products at fair prices.

## 👩‍💻 Developer
Name: Sakshi Jadhav  
Project Type: Academic Full Stack Project  

## 🎯 Objectives
- Connect farmers directly with customers
- Remove middlemen
- Provide fair pricing
- Promote digital agriculture

## 🛠 Technology Stack
- Frontend: React.js (Vite), Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB

## ⚙️ Project Structure
- api → Backend APIs & database
- client → Frontend UI

## ▶️ How to Run
1. Open project in VS Code
2. cd api → npm install → npm start
3. cd client → npm install → npm run dev
4. Open http://localhost:5173
